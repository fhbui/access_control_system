# access_control_system
## 📒概述

基于STM32F407+FreeRTOS+LVGL的智能门禁锁项目，能够通过密码、指纹、IC卡三种方式进行开门，同时通过UI交互可以对密码进行编辑，指纹、IC卡的信息进行增删。



## 🚀特点

- **UI交互设计**：通过LVGL构建了较为完整的UI，可以此进行密码操作和对指纹、IC卡信息进行增删处理；构建了各个模块到UI层的事件通知系统，模块的识别状态、结果会通过弹窗反映到UI层。
- 基于FreeRTOS操作系统，多种模块任务并发处理。



## 😔不足

- **UI设计简陋**：UI设计缺少现代产品设计美感。

- **代码耦合程度高**：驱动代码缺乏框架性，耦合程度较高；缺乏分层设计与面向对象设计思想。
